🧭 Side Quest: Blockbench
Path: Java
Dashboard Coding 3d Modeling Side Quest Tutorials Bedrock vs Java Assets Notebook AI Tools

Goal

Create a simple Shark model in Blockbench, export a PNG texture, and prep it for your mod. Earn +20 XP.

Checklist

  1. Open Blockbench (pinned on taskbar).
  2. New Java Entity → name it Shark.
  3. Add cubes to form the body, fins, and a launcher “barrel”.
  4. Auto-UV → Paint → export texture PNG.
  5. Save model (.bbmodel) to /public/models/ and PNG to /public/sprites/.



Use your art in-game

Copy your PNG to: src/main/resources/assets/sharkgun/textures/item/shark_gun.png (or your item id). Then run dev client.

Give command: /give @p sharkgun:shark_gun
